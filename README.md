# Movie Munch

Kid-friendly movie and TV rating app built as a local-first React + Vite web app.

## Stack

- React
- TypeScript
- Vite
- React Router
- localStorage

## Core Rules

- Local profiles on one shared device
- Title-level ratings only
- Score range `0-3`
- Multiple ratings over time
- Favorites do not affect averages or leaderboards
- Group average uses each user's latest rating for a title

## Main Files

- `src/`
- `public/`
- `package.json`
- `.replit`

## Run

```bash
npm install
npm run dev
```
